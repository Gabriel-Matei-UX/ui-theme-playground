# UI Theme Playground

## Overview
UI Theme Playground is an interactive tool designed to demonstrate how a complete and scalable design system can be generated using HSL color logic.

The project showcases how a single base color can dynamically generate an entire UI theme, including variations for backgrounds, borders, text, and components.

It is built as a visual and practical foundation for AI-assisted UI generation and design system scalability.

## Key Features

- 🎨 Dynamic theme generation using HSL color system
- ⚡ Real-time UI updates based on user input
- 📦 Full palette preview (primary, secondary, light, dark variations)
- 🧩 Component showcase (cards, sections, forms, inputs, buttons)
- 📱 Fully responsive (mobile-first approach)
- 💾 Persistent user settings (LocalStorage)
- 🎛️ Customization panel:
  - Base color (HSL)
  - Spacing between sections
  - Border radius

## Technical Approach

The core of the system is based on:

- CSS Custom Properties (variables)
- HSL color transformations
- Derived color scales using opacity and lightness adjustments
- Real-time DOM updates via JavaScript

All theme variations are computed from a minimal set of base values:
- Hue
- Saturation
- Lightness
